# SHIVA AI 🤖(Smart Hybrid Intelligent Virtual Assistant)

A lightweight Jarvis-like AI assistant with voice interaction, offline + online LLM support.

## Features (Planned)
- Voice input (STT)
- Voice output (TTS)
- Offline & online LLM
- Tool calling (system, apps, web)
- Memory

## Tech Stack
- Python
- LLaMA / Mistral (offline)
- Whisper (STT)
- Coqui / Piper (TTS)

## Project Structure
